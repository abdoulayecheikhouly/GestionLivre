# GestionLivre
Application Spring Boot pour la gestion des livres avec association à des catégories et gestion de la suppression logique, configurée avec application.yml et utilisant Lombok.
Technologies
Java 17+

Spring Boot

spring-boot-starter-web

spring-boot-starter-data-jpa

spring-boot-starter-validation

mysql-connector-java

Lombok

MySQL

YAML Configuration
